IP访问控制
禁止前台访问的IP 列表:
留空为所有 IP 均可访问；
当用户处于本列表中的 IP 地址时将无法访问本站点前台所有页面。
每个 IP 一行，既可输入完整地址，也可只输入 IP 开头，例如 "192.168."(不含引号) 可匹配 192.168.0.0～192.168.255.255 范围内的所有地址；
允许访问"后台"的IP 列表:
留空为不限制；（您当前的IP地址是：{$GLOBALS[_J][client_ip]}）
只有当管理员处于本列表中的 IP 地址时才可以访问后台，列表以外的地址访问将无法访问，但仍可访问网站前端用户界面，请务必慎重使用本功能。
每个 IP 一行，既可输入完整地址，也可只输入 IP 开头，例如 "192.168."(不含引号) 可匹配 192.168.0.0～192.168.255.255 范围内的所有地址;
